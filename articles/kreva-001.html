KREVA 001

クラフト feat. ZORN

ZORNさんとのコラボ。技能を極めた者同士のモノづくり。

ページ作成日: 2026年7月3日
元にしたYouTubeリンク: https://www.youtube.com/watch?v=e8RvyjBj4cY
確認した動画: KREVA「クラフト feat. ZORN」MUSIC VIDEO（KREVA本人公式チャンネル）

「クラフト」という言葉を、ヒップホップの曲名として選ぶセンスに、まず立ち止まらされます。工芸、手仕事、あるいは丁寧に組み上げられたモノ。KREVAとZORNは、この言葉を、自分たちのラップそのものの比喩として差し出しているように聴こえます。曲を再生してすぐに感じるのは、技術の応酬というより、技術への信頼です。互いのフロウを疑わず、相手が積んできた時間をそのまま受け止めた上で、自分の持ち場を鳴らしていく。そういう落ち着きが、この曲の緊張感の底にあるように思えます。ラップという表現を、若さや勢いだけで語らず、長く続けてきた者同士の仕事として提示する。その姿勢に、東京で働いていた頃に出会った、寡黙に技術を磨き続けていた職人たちのことを思い出します。彼らは自分の腕前を声高に語ることをせず、ただ手を動かし続けていました。それでいて、同じ水準の技能を持つ者同士が顔を合わせると、言葉を交わさずとも通じ合うものがある。KREVAとZORNの関係にも、そうした職人同士の呼応に近いものを感じます。この記事では、この曲が生まれた背景と、そこに刻まれた音の設計を、自分自身の記憶と重ねながら読み直していきます。

3度目の共演がたどり着いた場所

「クラフト feat. ZORN」は2022年2月16日にリリースされ、アルバム『LOOP END / LOOP START (Deluxe Edition)』に収録されています。音楽ナタリーやBARKSの報道によれば、この曲はKREVAとZORNにとって3度目の共演でした。最初は2020年8月、ZORNのデジタルシングル「One Mic feat. KREVA」にKREVAが客演として参加する形で実現し、同年10月には今度はKREVAがZORNを迎えた「タンポポ feat. ZORN」を発表しています。BARKSの記事では、「クラフト」はそこから約1年4カ月を経ての再タッグと伝えられており、両者が「908 FESTIVAL」というイベントのステージをともにするなど、単発の企画ではなく継続的な関係の中でこの曲が形になったことがうかがえます。

興味深いのは、最初の「One Mic」がZORN主導、次の「タンポポ」がKREVA主導という具合に、互いのフィールドへ交互に招き合う形で関係が育っていった点です。呼ぶ側と呼ばれる側が固定されない関係は、どちらかがどちらかに従属する共演ではなく、対等な立場でモノを作り合う関係だということを示しています。3度目にしてようやく、双方が発起人となるのではなく「モノづくり」という共通のテーマを掲げ、対等な立場で1つの曲に取り組んだ、という流れを想像すると、この曲の成立過程そのものが、テーマである「クラフト」を体現しているようにも見えてきます。

1度きりの客演であれば、話題作りの域を出ないこともあります。けれど3度目ともなると、そこには互いの現在地を確かめ合うような意味合いが生まれてくるはずです。「モノづくり」というテーマをあえて選んだのも、この2人が単発のコラボではなく、継続的な創作の関係を築いてきたことの表れだと感じます。ミュージックビデオは、KREVA作品の常連である映像作家・大喜多正毅が監督を務めたことも報じられており、こうした制作陣との継続的な関係もまた、この曲を支える「クラフト」の一部だったのかもしれません。

東京にいた頃、同じ現場に何度も呼ばれる職人を見てきました。1度きりの依頼で終わる相手は数え切れないほどいますが、同じ相手から繰り返し声がかかるようになると、その関係の質はまるで違ってきます。相手の仕事の癖を知り、任せられる範囲を互いに広げていく。KREVAとZORNの3度目の共演にも、そうした「重ねてきた仕事だからこそ許される踏み込み方」があったのではないかと想像します。客演という関係は、1度目は互いの様子見であることが多いものです。それが3度目になると、様子見の段階はとうに終わり、むしろ相手の技量を前提にした踏み込んだ表現が可能になっているように聴こえます。

数字にならない評価と、鳴り続ける存在感

「クラフト feat. ZORN」は、単独のデジタルシングルとしてではなく、アルバム『LOOP END / LOOP START (Deluxe Edition)』の収録曲としてリリースされた曲です。そのため、この曲単体のオリコンシングルランキングのような数値は、今回確認できた範囲の資料には見当たりませんでした。一方で、収録元のアルバムはKREVAの作品歴の中でも高い評価を受けてきたシリーズであり、ファンやリスナーの間で語り継がれてきた楽曲だと伝えられています。チャートの数字として明確に語れない曲であっても、リリースから年月が経ったいまも、日本語ラップを扱う個人ブログや考察サイトで繰り返し取り上げられていること自体が、この曲の評価のかたちなのだと思います。派手な記録よりも、聴き手の記憶に残り続けることの方が、この曲のテーマである「クラフト」にはふさわしいのかもしれません。

磐田で家や土地の相談を受ける仕事をしていても、似たようなことを感じます。契約が成立した件数や、扱った物件の規模だけでは測れない仕事があります。相談者の言葉にならない不安を汲み取り、家族それぞれの立場を調整し、最後まで見届ける。そうした仕事の質は、数字の外側にしか残らないことが多いのです。それでも、その仕事を見ていた誰かが、何年か経ってからふと思い出し、また相談を寄せてくれることがあります。数字として記録されない信頼が、時間差で戻ってくる瞬間です。「クラフト」という曲が、リリースから年月を経てもなお語られ続けているのも、似たような時間差の中にあるのではないかと思います。

韻の密度が示す、譲れない領域

日本語ラップを専門に扱う考察サイトの分析によれば、この曲は韻の密度と組み合わせの精度が高く評価されており、KREVAとZORNそれぞれが自分の持ち味を保ったまま、同じテーマの中で言葉を積み上げていく構成になっていると指摘されています。聴いていると、2人のフロウは決して同化しない、と感じます。KREVAの言葉運びは端正で、一語一語の置き場所に迷いがないように聴こえる一方、ZORNのフロウは体温がやや高く、言葉のはみ出し方に勢いがあるように聴こえます。異なる質感を持つ2人が、同じビートの上でぶつからずに共存している。それは、互いの領域を尊重しながら、自分の持ち場を譲らない、職人同士の仕事の進め方に近いもののように思えます。トラックの作り自体も、派手な展開で聴かせるというより、低く沈んだビートの上でラップそのものの質感を前面に出す設計になっていると感じます。この禁欲的とも言える鳴らし方が、「クラフト」という曲名の説得力を支えているのではないでしょうか。

細部への執着という点でも、この曲は徹底しているように聴こえます。「神は細部に宿る」という言葉がしっくりくるほど、音の置き方、間の取り方、韻の踏み方の1つ1つに、聴き逃してしまいそうな工夫が積み重なっているように感じられます。1度聴いただけでは気づかない仕掛けが、繰り返し聴くたびに見つかっていく。そういう作りの曲は、消費されて終わるのではなく、聴き手の側にも技術を要求してきます。良い仕事の現場に立ち会うと、同じような感覚を覚えることがあります。表面だけを見ていては気づかない工夫が、細部に何層にも重ねられていて、それに気づけるかどうかは、こちらの経験値にもかかっている。この曲を聴き込むという行為自体が、こちらの耳という技術を試されているようにも感じます。

技能を突き詰めた者同士が同じ場に立つと、そこには競争とは少し違う何かが生まれるように思います。相手の技量を認めた上で、自分の技量を差し出す。そういう関係性が、この曲のビートの隙間から伝わってくるように聴こえます。芸術性と、聴かれるための商業的な工夫との間で揺れるテーマが歌詞に織り込まれているという指摘もありますが、その葛藤自体を隠さずに提示する姿勢もまた、この曲がまとう「クラフト」らしさなのだと思います。迎合と本質のあいだで悩みながらも、最終的には自分の技術を信じて言葉を置いていく。その往復運動が、曲全体の緊張感を支えているように聴こえます。

磐田で磨く、それぞれのクラフト

介護や不動産の相談という仕事を続けていると、この仕事もまた、ひとつのクラフトなのだと感じることがあります。制度や法律の知識は基礎に過ぎず、実際に人と向き合う中で身につけていく技術の方が、はるかに時間がかかります。家を手放すという決断に至るまでの家族の迷いに寄り添う技術、土地の権利関係を整理しながら関係者全員の納得を探る技術。それらは資格や研修だけでは身につかず、現場で何度も試され、少しずつ磨かれていくものです。KREVAとZORNが、それぞれの現場でラップの技能を磨き上げてきたように、私もまた、この仕事における自分なりのクラフトを、日々少しずつ磨き続けています。

磐田という土地は、東京のように次々と新しい仕事が生まれる場所ではありません。同じ地域の同じ家々と、長い時間をかけて向き合い続けることになります。だからこそ、その分だけ、1つ1つの案件に費やせる技術の密度は濃くなるのかもしれません。派手さのない土地で、地味な仕事を積み重ねること。それもまた、「クラフト」という言葉がふさわしい営みなのだと、この曲を聴くたびに思います。

家族と過ごす時間の中でも、同じことを思います。子どもに何かを教えるとき、あるいは親の暮らしを気にかけるとき、そこには正解のマニュアルはなく、その都度、自分の持っている技術を総動員するしかありません。技能を極めた者同士が出会うと、そこには新しい何かが生まれる。この曲が持つ緊張感と充実感は、そうした出会いの豊かさを、リスナーである私たちにも分けてくれているように感じます。磐田という土地で、自分の仕事を積み重ねていくことも、遠く離れた場所で鳴らされたこの曲と、どこかで地続きなのだと思います。

受け継がれる技術と、次の世代への手渡し

KREVAとZORNの世代は違いますが、この曲を聴くと、年齢差を超えて技術がまっすぐに手渡されているように感じます。年下だから教える、年上だから教わる、という単純な構図ではなく、互いの得意分野を持ち寄って、その場でしか生まれない何かを作っている。仕事の現場でも、同じような場面に出会うことがあります。介護の仕事で長く現場に立ってきた人と、若い担い手が、それぞれの経験と体力を持ち寄って、1つのケースに向き合う瞬間です。年季の入った判断力と、新しい柔軟さが噛み合ったとき、どちらか1人では届かなかった質の仕事が生まれます。「クラフト」という曲が鳴らしているのも、そうした技術の受け渡しの豊かさなのだと思います。土地に根ざした仕事を続けていく中で、自分がいつか誰かにその技術を手渡す番が来ることも、この曲を聴くたびに、静かに意識させられます。
